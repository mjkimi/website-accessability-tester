More about this testing method
Accessibility Testing is the QA practice of confirming that the website is usable for as many people as possible, including people with disabilities such as vision impairment, hearing problems and cognitive conditions. Testing is done according to the Web Content Accessibility Guidelines (WCAG) 2.1: W3C Recommendation.
This app scans the HTML code of the webpage and shows any found mismatches with W3C documentation.

For example, this tag has an empty 'alt' attribute:
<img src="example.jpg" alt=""/>. App will let you know about this.

Fixing these kind of bugs makes your website more precise & neat, especially for screen readers and other accessibility tools.
